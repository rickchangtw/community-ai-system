# 多 Agent 智慧管理系統

## 系統概述

基於 OpenWiki 知識庫的多 Agent 智慧管理系統，服務於 600 戶社區。

## 六個核心 Agent

| Agent | 角色 | 職責 |
|-------|------|------|
| admin | 總幹事 | 系統管理、協調、決策 |
| property | 物業管理 | 維修、供應商、空間 |
| security | 保全 | 存取控制、威脅偵測 |
| fire | 消防 | 火警偵測、滅火、疏散 |
| energy | 節能管理 | 負載平衡、能源優化 |
| notify | 通知中心 | 廣播、訊息路由 |

## 通訊協議

定義 Agent 間的通訊格式和事件類型。

## 事件驅動系統

基於事件的 Agent 間通訊機制。

## 狀態管理

共享狀態和狀態遷移規則。

## 測試結果

- 3 個場景、10 個訊息
- 全部通過驗證

## 檔案結構

```
multi-agent-system/
├── agents/           # Agent 配置
├── protocols/        # 通訊協議
├── events/           # 事件定義
├── state/            # 狀態管理
└── tests/            # 測試腳本
```
